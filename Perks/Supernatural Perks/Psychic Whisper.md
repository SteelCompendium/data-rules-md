## Psychic Whisper

|                         |                                                          |
| :---------------------- | :------------------------------------------------------- |
| **Keywords:** Psionic   | **Type:** Maneuver                                       |

**Effect:** You send a telepathic message to the target that takes 10 seconds or less to speak.
