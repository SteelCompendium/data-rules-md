*The target is thrown several seconds back through time, and gets to do it all again.* **Keywords:** Chronopathy, Psionic, Ranged **Type:** Maneuver

|                         |                              |
| :---------------------- | :--------------------------- |
| **Keywords:**           | **Type:**                    |
