# Steel (11 Clarity)

*The target's skin becomes chrome and tough. They become an unstoppable titanium titan.* **Keywords:** Metamorphosis, Psionic

|                                 |                            |
| :------------------------------ | :------------------------- |
| **Keywords:**                   | **Type:** Maneuver         |
| **Distance:** Self or Ranged 10 | **Target:** Self or 1 ally |
