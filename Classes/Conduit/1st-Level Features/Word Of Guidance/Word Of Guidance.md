# Word Of Guidance

*You invigorate an attacking ally with divine energy.* **Keywords:** Magic, Ranged

|                         |                      |
| :---------------------- | :------------------- |
| **Keywords:**           | **Type:** Triggered  |
| **Distance:** Ranged 10 | **Target:** One ally |
